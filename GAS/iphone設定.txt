=====================================
iOSショートカット作成手順
GitHub Actions を iPhone から実行する
=====================================

【準備】
- GitHub Personal Access Token (PAT) が必要
- 作成方法: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
- 必要な権限: repo または workflow

=====================================
手順
=====================================

1. iPhoneで「ショートカット」アプリを開く

2. 右上の「+」をタップして新規作成

3.「アクションを追加」をタップ

4. 検索で「URL」と入力 →「URLの内容を取得」を選択

5. 以下を設定:

   【URL】
   https://api.github.com/repos/epoch0718/py_timetree_scraper/actions/workflows/main.yml/dispatches

   【方法】
   POST

6.「ヘッダを追加」をタップして以下を追加:

   ヘッダ1:
   - キー: Authorization
   - 値: Bearer ghp_あなたのトークン
   
   ヘッダ2:
   - キー: Accept
   - 値: application/vnd.github.v3+json

7.「本文を要求」をタップ →「JSON」を選択

8.「新規フィールドを追加」→「テキスト」
   - キー: ref
   - 値: main

9. (任意) 結果通知を追加:
   - 「+」で別のアクションを追加
   - 「通知を表示」を検索して追加
   - テキスト: TimeTree実行完了！

10. 右上の「完了」または下矢印をタップ

11. ショートカット名を「TimeTree実行」に変更

=====================================
ホーム画面に追加
=====================================

1. 作成したショートカットを長押し
2.「詳細」または「ⓘ」をタップ
3.「ホーム画面に追加」をタップ
4. アイコンをカスタマイズ（任意）
5.「追加」をタップ

=====================================
使い方
=====================================

ホーム画面のアイコンをタップするだけ！
→ GitHub Actions のワークフローが実行されます

=====================================
トラブルシューティング
=====================================

エラーが出る場合:
- トークンが正しいか確認
- トークンの権限（repo または workflow）を確認
- URLが正しいか確認
- ヘッダの「Bearer 」の後にスペースがあるか確認
